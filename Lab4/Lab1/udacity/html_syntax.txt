1. Using the W3C HTML Validator(opens in a new tab), 
what happens when you enter the following HTML into the 
"validate by input" text box:
<h1>This is a title</h1>

answer: 
1. the validator found errors or warnings in the HTML

Now, modify the HTML you entered previously so it follows
the HTML template. Make sure to set doctype for HTML5!
What is the message displayed once the validator 
successfully validates your code?

answer:
